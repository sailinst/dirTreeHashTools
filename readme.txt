1、简介

   1)  用Go开发的目录检索工具。
   2)  对整个目录下的所有文件进行遍历，获取所有文件的大小和计算文件的sha1哈希值，记录在dirTreefile.txt文件里面。
   3)  结果文件格式：每一行一个文件，用逗号隔开，前面是文件名称，后面是哈希值，文件大小
   4)  可以指定忽略哪些文件类型，用户输入指定的文件类型，则不计入检索范围。
   5)  代码简洁，安全，高效。
   6)  要求支持go get, go run, go build命令

2、操作说明
   1)  启动程序：可以直接go run main.go启动程序，也可以go build 生成可执行文件启动。
   2)  程序启动后，会提示用户输入要检索目录的路径，回车后接着提示用户输入需要忽略的文件类型，即文件后缀名（输完一个后缀名回车，输入下一个后缀...）。
   3)  可以检索的文件后缀名全部存在 suffixArray[]中。
   4)  终端会打印出当前检索出的所有文件的路径，文件的sha1值，文件大小以及所有
   检索到的文件总数。
   5)  随即将这些数据写入 dirTreefile.txt 文件中。
   6)  用户提示操作： 输入'quit' or '退出'退出系统，否则继续下一次检索操作。

3、亮点
   1)  文件后缀名全部存在 suffixArray[]中，可以随时添加文件的后缀名。
   2)  用单链表存储suffixArray[]的值，方便删除操作(数组不能删除元素)。
   3)  用户输入要忽略的文件后缀名，用list.Remove()方法删除相关元素。

4、不足
   1)  每一次检索后的数据会写入同一个文件中，即dirTreefile.txt。下次操作会抹
   掉上一次的所有数据。所以，做完一次检索后需要注意保存数据。
   2)  ...
